UE和WorldCreator的地形互导

Posted on 2023-02-07 Edited on 2023-02-08
Symbols count in article: 2k Reading time ≈ 2 mins.
Landscape between UE5 and World Creator

存在的问题

尺寸标准不一致

  • WorldCreator的地形尺寸是2的n次幂（512, 1024, 2048 ... ），而UE的地形尺寸一般为奇数（参考官方文档的地形尺寸建议），以1017x1017尺寸，分块信息如下图所示的地形为例。WorldCreator导入UE高度图进行编辑后，只能输出1024x1024的高度图。在UE导入WorldCreator高度图时会裁剪回1017x1017的尺寸，导致边缘高度丢失。迭代多次后，边缘高度信息将大量丢失。
UE Landscape Info

地形边缘高度丢失

  • WorldCreator不支持长宽不等的地形。
长宽不等的地形在WorldCreator中失真

解决方法：

  • 对于需要进入WorldCreator编辑的地形，应保持长宽等尺寸。
  • UE输出高度图时匹配WorldCreator的地形尺寸。

    1. 根据WorldCreator高度图的裁剪方式（以左上角为枢轴点，裁剪右下方的区域），将UE的地形扩展至2的n次幂（延X轴和Y轴的正方向），匹配WorldCreator的地形尺寸

    2. 在WorldCreator处理完地形后，导出高度图时将其裁剪为UE原地形的尺寸。

WorldCreator高度图的裁剪方式

扩展UE的地形尺寸至2的n次幂

高度缩放不匹配

对于一张16位的灰度图（在默认缩放值下），UE地形的高度范围是-256米到256米，可以描述高度差在512米以内的地形。WorldCreator的地形范围是-106到106米。

Heightmap （16bit, Single Channel）

Height Range Difference

解决方法：

  1. 在WorldCreator导入高度图时，将高度缩放值修改为2.4151（512/212），还原UE地形的高度。

    WorldCreator 高度缩放


  2. 在WorldCreator导出高度图时，开启自动归一化选项，将地形高度映射为0~255的灰度范围。

  3. 在UE导入WorldCreator的高度图后，根据高度范围计算地形的高度缩放和偏移值。

高度图导入/导出流程

由于UE和WorldCreator的地形存在上述问题，在导入和导出高度图的过程中需要计算地形在不同平台的高度缩放值和偏移值。HDA（heightmap_worldcreator_yh_SOP）通过读取UE地形，在后台完成相关的计算并将结果显示在参数面板中，如下图所示：

HDA参数面板

  • Min Height / Max Height：地形的高度范围
  • UE Landscape
    • Offset Z：UE地形的高度偏移值
    • Scale Z：UE地形的高度缩放值
    • Heighmap File：UE高度图的输出路径
    • Save Heightmap：输出UE高度图
  • World Creator Terrain
    • Height Scale：WorldCreator 的高度缩放值
    • Precision (Meter)：World Creator 的地形精度

UE导出高度图

  1. 将HDA（heightmap_worldcreator_yh_SOP）拖入场景中（Transform数据需要归零）

    HDA Transform


  2. 设置HDA的输入类型（Landscape Input）并选择目标地形（不能选择地形分块）

    HDA输入


  3. 点击HDA参数面板的Read Landscape获取地形数据

  4. 设置HDA参数面板的高度图输出路径Heightmap File，然后点击Save Heightmap输出高度图

WorldCreator导入高度图

WorldCreator导入高度图

  1. 根据地形尺寸设置地形精度（GLOBAL → Precision），对应HDA参数面板中的Precision (Meter)

    Size Precision
    8912 x 8912 1 / 8 Meter
    4096 x 4096 1 / 4 Meter
    2048 x 2048 1 / 2 Meter
    1024 x 1024 1 Meter
    512 x 512 2 Meter
    256 x 256 4 Meter

  2. 将网格细分设置为10（CUSTOM BASE SHAPE → Subdivision）

  3. 设置高度缩放值（CUSTOM BASE SHAPE → Height Scale）

  4. 导入UE输出的高度图

  5. 编辑地形

  6. 导出高度图

    • 16 Bit, Single Channel
    • Auto Normalization
    • 裁剪尺寸（Export Area）

    WorldCreator导出高度图


UE导入WorldCreator的高度图

UE导入WorldCreator高度图

  1. 在地形模式下（Landscape → Manage → Import），读取WorldCreator输出的高度图，点击Import更新地形.
  2. 根据HDA参数面板的Scale Z和Offset Z设置地形的高度缩放值和位移
  • 将地形的缩放值（Transform→Scale.z）设置为Scale Z
  • 将地形的Z轴位移（Transform→Location.z）加上Offset Z
